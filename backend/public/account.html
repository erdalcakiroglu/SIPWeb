Offline .lic Download

Download a device-bound license file from the active records below, then import it in the desktop app.

Settings > License > Import .lic
Use the same Device ID that SQL Performance Intelligence™ shows on the target machine.

The exported file is tied to that device. If the device fingerprint changes, download a new `.lic` file for the updated Device ID.

  1. Open SQL Performance Intelligence™ on the machine where you will use the license.
  2. Go to Settings > License.
  3. Select Import .lic.
  4. Choose the downloaded file and confirm the import.
